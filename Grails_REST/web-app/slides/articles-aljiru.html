Sistema de gestión de paquetes

En la década de los 90 Roy T. Fielding realizó un estudio en el que buscaba el estilo arquitectónico que más propiedades aportara a sistemas en Red basados en Hypermedia.

El estilo más prometedor fue Layered-Code-on-Demand-Client-Cache-Stateless-Server (LCODC$SS).

Dependencias de los paquetes

Cada paquete contiene una lista de dependencias y una lista de sugerencias

dpkg-query

Es una de las muchas herramientas de gestión de paquetes que se pueden encontrar en linux

Sirve para consultar la base de datos de paquetes

~$ dpkg-query -l zlib1
